# Начинаем работу с QR-Passport
Эта инструкция – ваш помощник для начала работы. Интерфейс системы будет выглядеть по-разному в зависимости от прав доступа (_подробнее о правах доступа см. [пользователи](company/users.md#anchor)_).

{% note info %}

Все примеры в инструкции носят справочный характер. 

{% endnote %}

{% note info %}

При первом знакомстве с системой QR-Passport рекомендуется начать с раздела [Быстрый старт](quick_start.md) для создания тестовой модели. Для тех, кто уже знаком с системой, предусмотрены следующие разделы инструкции для более глубокого изучения всех возможностей. 

{% endnote %}

Для начала работы с QR-Passport войдите в систему по [ссылке](https://qrpassport.app/admin). Используйте для входа имя пользователя и пароль, которые были отправлены на вашу электронную почту.
