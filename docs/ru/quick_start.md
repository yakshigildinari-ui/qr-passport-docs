# Быстрый старт

Раздел содержит пошаговую инструкцию по освоению основного процесса работы в системе QR-Passport – создания моделей оборудования и регистрации изделий. Практический пример поможет быстро освоить ключевые операции платформы.

## Цель
- [ ] Освоение процесса создания моделей оборудования
- [ ] Изучение процедуры регистрации изделий
- [ ] Получение практического опыта работы с системой

## Порядок работы
Инструкция разделена на логические этапы, выполнение которых позволяет последовательно освоить функционал системы. Каждый этап содержит четкие шаги и пояснения. Для начала работы рекомендуется ознакомиться с основными понятиями системы и перейти к заданию.

## Основные понятия
* **Модель оборудования** –  типовая конфигурация, которая определяет общие характеристики для группы однотипных устройств. Содержит общие характеристики, документацию и динамические поля для всех относящихся к ней изделий.
* **Изделие** – конкретная единица продукции, созданная на основе определенной модели. Наследует общие характеристики модели, но имеет уникальные параметры.
* **Динамические параметры** – настраиваемые характеристики, которые добавляются к модели и заполняются для каждого изделия отдельно. Позволяют учитывать индивидуальные особенности разных единиц оборудования.
* **Зарегистрировать изделие** – это процесс внесения в систему данных о конкретной единице продукции, который включает заполнение динамических параметров (серийный номер, дата изготовления и другие). Результат: изделие получает собственный QR-код и становится полноценным объектом учета в системе с возможностью отслеживания его характеристик, документации и возможностью технической поддержки.

## Задание
### Описание
Создать с нуля модель пневмопривода серии VT с наименованием **VT065 S07 F05/07 14 P5021 LLT НЗ** в системе QR-Passport, зарегистрировать 5 изделий этой серии с датой изготовления 11/2025 и серийными номерами:
* 112025.001;
* 112025.002;
* 112025.003;
* 112025.004;
* 112025.005.

Скачать паспорта и QR-коды на эти 5 изделий.
Компания-изготовитель этих пневмоприводов – VTORK Technology (Wuxi) Co., LTD.

### Роль в системе
Для данной задачи ваша роль в системе _company administrator_ с полными правами доступа к системе, кроме доступа к разделу **Техническая поддержка**.

### Структура оборудования
В первую очередь необходимо создать структуру оборудования. 
Для этого выполните следующие шаги:

1. В открывшимся окне перейдите в подраздел **Группы** 
2. Нажмите кнопку **Добавить группу**
3. Введите название группы `[Пневмоприводы]`(*группа_пневмо) – так назовем нашу родительскую группу, т.к. это название нашего оборудования и его можно взять за основу структуры.
4. Нажмите кнопку **Сохранить**

Чтобы конкретизировать, что наша модель относится не только к группе _пневмоприводы_, но и к _серия VT_, необходимо создать внутри родительской группы – дочерную. 

Для этого выполните следующие шаги:

5. Нажмите на наименование группы **Пневмоприводы**, откроется окно для редактирования
6. Нажмите кнопку **Редактировать**
7. В блоке **Группы** нажмите кнопку **Добавить еще одну группу**, всплывет поле для редактирования
8. Введите наименование подгруппы [`Серия VT`](*подгруппа_серия), описание типа продукции заполняется по желанию
9. Нажмите кнопку **Сохранить**

10. Перейдите в подраздел **Производители**
11. Нажмите кнопку **Добавить производителя**

### Модель оборудования
После создания структуры для нашей модели приступаем к созданию самой модели. Для этого выполните следующие шаги:

14. Перейдите в конечную подгруппу, где будет находится модель, в нашем случае это **Серия VT**, чтобы раскрыть список групп нажмите **→**
15. Нажмите кнопку **Редактировать**
16. В блоке **Модели оборудования** нажмите кнопку **Добавить еще одну модель оборудования**, откроется окно создания модели
17. Заполните поля этими данными:
    * _Название_ – [`VT065 S07 F05/07 14 P5021 LLT НЗ`](*наим_модели) 
    * _Изображение_ – добавьте скачанное изображение [скачать][2]
    * _Описание_ – [`Привод пневматический четвертьоборотный, одностороннего действия. Присоединительный фланец F05/07 по ISO 5211, квадрат штока 14 мм. Климатическое исполнение УХЛ1 (-60°С...+80°С). Рабочий диапазон давления управляющего воздуха 3-8 бар.`](*описание)
    * _Производитель_ – нажмите кнопку **Выбрать**, чтобы выбрать производителя, которого мы создали выше
    * _Группа_ – группа должна быть уже выбрана, т.к. мы перешли к созданию модели через подраздел **Группы**. Это поле нужно выбирать, если мы создаем модель через подраздел **Модель оборудования**
    * _Основная гарантия_ – вводим количество месяцев основной гарантии [`36`](*мес)
    * _Технический паспорт_ – добавьте скачанный паспорт [скачать][3]
    * _Инструкция по эксплуатации_ – добавьте скачанную руководство по эксплуатации [скачать][4]
    * _Динамические параметры_ – создайте два динамических параметра ([`Серийный номер`](*динам_поле1), [`Дата изготовления`](*динам_поле2)), нажав кнопку **Добавить новый**
    * _Документы_ – добавьте скачанный сертификат [скачать][5]
18. Нажмите кнопку **Сохранить**

### Регистрация изделий
После создания модели, переходим к регистрации изделия в системе QR-Passport. Для этого выполните следующие шаги:

19. Перейдите в подраздел **Модели оборудования**
20. Нажмите на наименование модели **VT065 S07 F05/07 14 P5021 LLT НЗ**
21. Перейдите к вкладке **Изделия**
22. Нажмите кнопку **Массовая регистрация**
23. Заполните поля этими данными:
    * _Серийный номер_ – [`112025.00{1}`](*серийный_номер)
    * _Дата изготовления_ – [`11/2025`](*дата_изготовления)
    * _Инкремент_ – [`1`](*инкремент)
    * _Количество изделий_ – [```5```](*количество)
24. Нажмите кнопку **Сохранить**

{% note info "" %}

Подробнее о регистрации изделий см. [регистрация изделий](equipment/models.md#anchor)

{% endnote %}

### Скачивание QR-кодов

25. Перейдите в подраздел **Зарегистрированные изделия**
26. Галочками выберите серийные номера изделий (112025.001-112025.005)
27. В выпадающем списке выберите **Скачать QR-коды**
28. Нажмите кнопку **Выполнить**
29. Выберите формат этикетки – квадрат
30. Выберите размер этикетки – 30×30
31. Нажмите кнопку **Скачать QR-код**

34. В выпадающем списке выберите **Скачать технические паспорта**
