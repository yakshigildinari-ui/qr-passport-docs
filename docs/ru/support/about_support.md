# О разделе

{% note info "" %}

Доступ к этому разделу предоставляется по подписке _«Премиум»_.

{% endnote %}

Раздел **Техническая поддержка** предназначен для обработки и управления запросами от клиентов. Функционал позволяет регистрировать входящие заявки, отслеживать их статус и обеспечивать решение инцидентов.
Он включает несколько подразделов:
1. [Тикеты](support.md#anchor)
2. [Категории тикетов](ticket_categories.md#anchor)
